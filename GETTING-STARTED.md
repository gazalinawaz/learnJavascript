After completing several lessons, work on the projects in the `projects/` folder to apply what you've learned.

## 💡 Learning Tips

### 1. Code Every Day
Even 30 minutes daily is better than 3 hours once a week. Consistency is key.

### 2. Type, Don't Copy
Type out every example yourself. Muscle memory helps learning.

### 3. Experiment
Break the code. Change values. See what happens. This is how you truly learn.

### 4. Use Console.log()
Print everything! Understanding what's happening at each step is crucial.

```javascript
let x = 5;
console.log("x is:", x);

let y = x * 2;
console.log("y is:", y);
```

### 5. Debug Actively
When something doesn't work:
- Read the error message carefully
- Check your syntax
- Use console.log to track values
- Google the error message
- Ask for help if stuck for more than 30 minutes

### 6. Build Projects
Theory is important, but building projects is where real learning happens.

### 7. Review Regularly
Come back to previous lessons. Repetition strengthens understanding.

## 📝 How to Run Your Code

### Method 1: Node.js (Recommended)
```bash
node filename.js
```

### Method 2: Browser Console
1. Open browser (Chrome/Firefox)
2. Press F12 to open DevTools
3. Go to Console tab
4. Copy and paste code
5. Press Enter

### Method 3: HTML File
```html
<!DOCTYPE html>
<html>
<head>
    <title>JavaScript Test</title>
</head>
<body>
    <h1>Check the console (F12)</h1>
    <script src="your-file.js"></script>
</body>
</html>
```
